# 🚀 Space Shooter Game

A 2D space shooter game built with WPF and Visual Basic .NET in Visual Studio.

## 🎮 How to Play
- ⬅️ ➡️ Arrow keys to move
- Space to shoot
- Avoid enemy ships and their bullets
- R to restart after game over

## ✨ Features
- Enemy ships that spawn and move towards you
- Enemies that aim and shoot directly at you
- Score and high score tracker
- 3 lives system
- Enemy death animation
- Sound effects for shooting, explosions and game over
- Starfield background

## 🛠️ Built With
- Visual Basic .NET
- WPF (Windows Presentation Foundation)
- Visual Studio
